# Анализ удержания клиентов Яндекс Афиши

## 📌 Описание проекта
Проект посвящён анализу поведения пользователей сервиса «Яндекс Афиша».  
Цель — выявить факторы, влияющие на повторные покупки и рост числа заказов, а также предложить маркетинговые рекомендации.

## 📁 Структура репозитория
- `main.ipynb` — Jupyter Notebook с полным анализом, визуализациями и выводами.
- `requirements.txt` — список используемых библиотек Python.
- `.env.example` — пример файла с переменными окружения (для подключения к БД).
- `.gitignore` — перечень файлов, исключённых из Git.
- `README.md` — описание проекта (текущий файл).
